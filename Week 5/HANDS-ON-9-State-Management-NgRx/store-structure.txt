NgRx Flow
=========

1. Component dispatches action:
   this.store.dispatch(loadCourses())

2. Effect listens for action:
   loadCourses$ effect → HTTP call

3. Effect dispatches success/failure action:
   loadCoursesSuccess({ courses }) or loadCoursesFailure({ error })

4. Reducer updates state (immutable):
   { ...state, courses, loading: false }

5. Selector emits new value:
   selectAllCourses → courses array

6. Component receives via async pipe:
   courses$ | async

Key Principles
==============

Actions:
- Describe events (past tense: "Courses Loaded")
- Contain minimal data (no transformation)
- Tagged with feature prefix: [Course]

Reducers:
- Pure functions (no side effects)
- Immutable state updates (spread operator)
- Handle one action at a time

Selectors:
- Memoized (performance optimization)
- Composable (build complex from simple)
- Encapsulate state shape

Effects:
- ONLY place for side effects
- Listen to actions, perform async work
- Dispatch new actions with results

State:
- Single source of truth
- Immutable (never mutate directly)
- Normalized (avoid duplication)
